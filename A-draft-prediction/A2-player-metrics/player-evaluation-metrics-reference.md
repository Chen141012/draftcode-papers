# NBA 球员价值评估指标速查

> 比赛用 26 年历史数据预测选秀，需要理解球员评估指标的含义和适用场景。
> 这篇只列**比赛模型可能用到的指标**，不深入公式推导。

## 指标分类

### 第一类：传统基础数据（所有数据集必然包含）

| 指标 | 缩写 | 说明 | 预测价值 |
|------|------|------|---------|
| 场均得分 | PPG | 基础得分能力 | 中等 |
| 篮板 | RPG | 内线/锋线评价基础 | 中高 |
| 助攻 | APG | 组织能力 | 中等 |
| 投篮命中率 | FG% | 得分效率 | 高 |
| 三分命中率 | 3P% | 空间能力（当前NBA最关键指标之一） | 非常高 |
| 罚球命中率 | FT% | 投射基本功 | 中高 |
| 抢断/盖帽 | SPG/BPG | 防守贡献 | 中等 |
| 失误 | TOV | 决策质量 | 中等 |

### 第二类：进阶效率指标

| 指标 | 说明 | 计算方法 | 适用 |
|------|------|---------|------|
| **PER** (Player Efficiency Rating) | 每分钟综合效率，15=联盟平均 | 所有正面贡献-负面贡献，按分钟标准化 | 快速对比，但有偏向进攻球员的争议 |
| **TS%** (True Shooting) | 真实命中率，考虑两分/三分/罚球权重 | PTS / (2 * (FGA + 0.44 * FTA)) | 评估得分效率的最佳指标 |
| **eFG%** (Effective FG%) | 有效命中率，调整三分权重 | (FGM + 0.5 * 3PM) / FGA | 快速评估投射效率 |
| **WS** (Win Shares) | 预估球员为球队带来多少胜场 | 基于进攻/防守胜场贡献分解 | 历史数据充足，适合回顾性分析 |
| **WS/48** | 每48分钟胜场贡献 | WS / (Min / 48) | 效率标准化，跨球员对比 |

### 第三类：高阶混合指标（比赛时可能遇到的概念）

| 指标 | 类型 | 数据依赖 | 说明 |
|------|------|---------|------|
| **BPM** (Box Plus-Minus) | 纯基础数据 | 仅box score | 只需要基础统计数据就能计算，历史数据完整 |
| **RAPM** (Regularized APM) | 混合 | 需要on-off数据 | 当前黄金标准，但需要play-by-play数据 |
| **RAPTOR** | 混合 | 需要player tracking | FiveThirtyEight的指标，含球员追踪数据 |
| **EPM** (Estimated Plus-Minus) | 混合 | 需要player tracking | 被多名NBA高管称为"gold standard" |
| **LEBRON** | 混合 | 需要on-off数据 | BBall-Index的Luck-adjusted RAPM |

> **比赛场景注意**：26 年历史数据大概率只包含基础数据 + box score（PER/TS%/BPM 这级）。
> Player tracking 时代的指标（RAPTOR/EPM）2013年后才有，历史数据可能没有。

## 哪些指标对选秀预测最有用

基于学术论文和实际研究：

| 优先级 | 特征 | 理由 |
|--------|------|------|
| ⭐1 | **3P% + 三分出手数** | 空间能力是现代NBA最值钱的技能，历史数据证明高3P%大学球员的NBA成功率更高 |
| ⭐2 | **得分效率 (TS%/eFG%)** | 比单纯看PPG更有预测力，能识别高效得分手 |
| ⭐3 | **体型数据 (身高/体重/臂展/站立摸高)** | 体测数据对选秀顺位预测极强，尤其是前锋/C位置 |
| ⭐4 | **年龄** | 年轻球员(大一/19岁)的NBA成功率显著高于大四毕业生，但需要区分"年轻有潜力"vs"成熟即战力" |
| ⭐5 | **篮板率/助攻率 (进阶版)** | 标准化后的效率指标比原始计数更有预测力 |
| ⭐6 | **所在联盟强度 (conference)** | Power 5 (ACC/Big 12/SEC/B1G/Pac-12) 出身球员的命中率显著更高 |

## 关键数据源

| 网站 | 说明 |
|------|------|
| basketball-reference.com | 最完整的NBA历史数据，PER/BPM/WS/TS%全有 |
| sports-reference.com/cbb | NCAA大学篮球数据，与bbref同体系 |
| nbarapm.com | RAPM历史数据可视化 |
| dunksandthrees.com | 指标对比分析平台 |

## 比赛日速查

```
数据到达后，优先计算：
1. TS% = PTS / (2 * (FGA + 0.44 * FTA))
2. 年龄 = draft_year - birth_year (如果birth year在9月后-1)
3. 体型指标: 如果包含体测数据，优先使用wing span和standing reach
4. 联盟强度 dummy: Power 5 = 1, 其他 = 0
```
